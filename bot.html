How it works

Open mandate.
Open trade log.

Three rules govern the experiment. They're public so you can hold the bot to them — and so it doesn't quietly become an undisclosed recommendation service.

01
RuleMandate

Paper account, intraday only

The bot trades a simulated $100k account using real-time market data. No overnight positions. No leverage above 2×. No real money is allocated.

02
RuleLogging

Every trade, every reason

Each fill is published with timestamp, instrument, size, P&L, and a one-paragraph reasoning trace from the agent. Wins and losses both. Edits get an erratum.

03
RuleHonesty

If it underperforms, we say so

The point isn't to demo a winner. The point is to document what an LLM-driven agent actually does in a live market — including the boring losses and the spectacular ones.
